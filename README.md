# Get started
## Clone
- ```git clone https://github.com/rxn7/chess --recurse-submodules```

## Setup
- ```cmake -Bbuild -GNinja``` I like using ninja, you can use any other generator

## Compile
- ```cmake --build build```

## Run
- ```pushd build ; ./chess ; popd```
