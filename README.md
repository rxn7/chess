# Get started
## Clone
- ```git clone https://github.com/rxn7/chess --recurse-submodules```

## Setup
- ```./configure.sh```

## Compile
- ```./build.sh``` - Debug version
- ```./build.sh -r``` - Releaase version

## Run
- ```./run.sh```- Debug version
- ```./run.sh -r```- Release version
